About Me

As a web developer with over five years of experience, I initially dipped my toes into web development during a project course at UMass Lowell while earning my Bachelor in computer science. I was instantly hooked and sought to create a new website for Georgetown Community Television, learning PHP and SOAP along the way to interface with their database and scheduling server. As a result of this work, Rowley Community Media brought me on to create a website for them from the same codebase with tweaks for their branding. Both sites are currently undergoing a major transition from the old SOAP API to a RESTful API thanks to new video servers with enhanced capabilities that include Video On Demand and live streaming.

Some hobby projects are also included here, mainly small utility pages for working with AviSynth scripts and creating chapter files for mp4 videos. More recent projects are being added from UCF's Coding Boot Camp to stay up to date on current coding technologies and best practices.
